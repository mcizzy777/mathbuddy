Area Calculators

Calculate the area of different geometric shapes using the formulas below.

Rectangle

Formula: A = Length × Width

Square

Formula: A = Side²

Triangle

Formula: A = ½ × Base × Height

Circle

Formula: A = πr²

Parallelogram

Formula: A = Base × Height

Trapezoid

Formula: A = ½ × (Base₁ + Base₂) × Height
